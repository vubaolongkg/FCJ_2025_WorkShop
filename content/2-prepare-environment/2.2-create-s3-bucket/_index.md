## Create S3 Bucket

1. Go to **S3** from the AWS Console.
2. Click **Create Bucket** and name it (e.g., `my-data-pipeline-bucket`)
3. Inside the bucket, create two folders:
   - `raw/` – for raw input data
   - `processed/` – for cleaned/transformed data
