## 4.3 Run Job & Verify Output

1. Go to Glue → Jobs → select `ProcessJob` → **Run**
2. Monitor job status: `RUNNING → SUCCEEDED`
3. Open S3 Bucket → check the `processed/` folder
4. Verify output file format: Parquet or CSV
