7.3 Attach S3 Trigger

At Lambda Function, select Add Trigger

  • Trigger Configuration: S3
  • Bucket: s3/sales-data-bucket-2025
  • Event types: All Objects create events Then press Add Trigger Test Lambda and Trigger
  • Add new files to S3
  • salesdatacrawler started run
  • salesdatacrawler finished running Configure Amazon EventBridge so that other actions are also triggered after crawling data
  • Go to Amazon EventBridge home page
  • Select Create rule
  • Select Event Pattern as JSON
{
"detail": {
"crawlerName": ["salesdatacrawler"],
"state": ["SUCCEEDED"]
}
}

  • Then configure as shown, role is LambdaRole Test again, after the first Crawl is finished, ETL will automatically run Job
